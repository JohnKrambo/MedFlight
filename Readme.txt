Hier kommt einmal die Doku zu unser genialen APP rein ;)

#Struktur#######
Die App sollte sich nur auf eine Seite beschränken auf der möglichst wenige Knöpfe vorhanden sind damit
der Nutzer sich im Notfall nicht verklicken kann.
########

#Javascript Libary#####
Als Javascript Libary wird zepto benutzt. Hierbei handelt es sich um eine Phonegap kompatible 
Lib die von der Nutzung sehr stark an Jquery erinnert und damit ausserdem Kompatibel ist. Der Vorteil ist, dass
die Libary nur eine Kleine .js datei groß ist und über alle benötigten funktionen verfügt.

Zur Installation der App muss der QR code mit einem Androidphone gescannt werden, 
die Installation läuft dann von alleine. 
Der QR-code ist zufinden unter dem Ordner inst.
